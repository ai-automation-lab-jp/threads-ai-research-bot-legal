プライバシーポリシー

本ポリシーは、データ処理の法的根拠や適用法令への適合を断定するものではありません。実際の運用にあたっては、必要に応じて運営者が専門家へ相談のうえ内容を確定してください。

1. 運営者

AI副業自動化ラボ（個人運営）

2. 対象サービス

Threads AI副業リサーチBOT（Threads公式APIを利用した、個人運用の市場調査支援ツール）

3. 取得する情報

Threads公式APIを通じて、以下の情報のみを取得します。

  • 公開Threads投稿
  • 投稿ID
  • 投稿本文
  • 投稿日時
  • permalink（投稿URL）
  • username（投稿者のユーザー名）
  • media_type（メディア種別）
  • 検索に使用したキーワード
  • Threads公式APIが返す、取得可能な公開フィールド
  • 投稿本文から抽出したURL
  • 抽出したURLの導線分類結果
  • 訴求語句の検出結果
  • 優先度スコア（機械的なルールベースで算出）
  • 重複投稿の判定結果

4. 取得しない情報

  • Threadsのパスワード
  • Instagramのパスワード
  • Facebookのパスワード
  • 非公開投稿
  • ダイレクトメッセージ（DM）
  • LINE内部の情報（トーク内容等）
  • 有料note等の本文
  • PDF本文
  • 利用者端末内の個人ファイル
  • 正確な位置情報
  • 決済情報
  • 投稿者の信用スコア
  • 投稿内容の真偽判定結果
  • 個人の信用評価

5. 情報の取得方法

Threads公式API（公開投稿の検索機能）を通じてのみ情報を取得します。非公式な手段、Webスクレイピング、ブラウザ自動操作は使用しません。

6. 利用目的

取得した情報は、以下の目的にのみ利用します。

  • AI副業関連の公開投稿・訴求手法・外部導線の傾向を、人間が分析するための一次スクリーニング
  • 手動分析の優先順位付けのための優先度スコアの算出（機械的なルールベース集計であり、投稿内容の真偽・投稿者の信用度・収益性を判定するものではありません）

取得した情報を、上記目的以外（第三者への販売・提供、広告、自動投稿・自動アクション等）に利用することはありません。

7. 保存場所

取得した情報は、運営者が管理するローカル環境のファイルシステムにのみ保存します。外部のデータベースやクラウドサービスは使用していません。GitHubリポジトリへ実データをコミットすることもありません。

8. 保持期間

取得・生成したデータは、以下を保存上限の目安として扱います。

データ種別ごとの保持期間（保存上限）
データ種別保持期間（上限）
raw JSON取得日から最大90日
processed JSON取得日から最大180日
人間向けJSON取得日から最大365日
CSV取得日から最大365日
Markdownレポート取得日から最大365日
ログ生成日から最大90日
アクセストークン有効期限、失効、連携解除、または不要となるまで

上記はいずれも保存上限であり、無期限保存や永久保存を行うものではありません。現段階では自動削除の仕組みは実装されておらず、運営者が上記の上限を目安に定期的に保存状況を確認し、手動で削除します。削除依頼があった場合も、下記「12. 削除要求」に基づき対象を確認したうえで手動で削除します。

なお、法令上の対応またはセキュリティインシデント対応等のために必要と判断される場合、上記の上限を超えて必要最小限の期間データを保存する可能性があります。この場合であっても、特定の法令への適合性を断定するものではありません。

9. 第三者提供

取得した情報を第三者へ販売・提供・共有することはありません。

10. 外部サービス

本ツールはThreads公式API以外の外部サービスと連携していません。本サイト自体も、外部スクリプト・外部フォント・アクセス解析・広告等の外部リソースを一切読み込みません。

11. セキュリティ対策

  • アクセストークン等の認証情報は、APIリクエストのAuthorizationヘッダー経由でのみ送信します。
  • アクセストークン等の認証情報を、ソースコード・ログ・成果物・本サイトへ記載することはありません。
  • 取得した情報は、Git管理対象外のローカルディレクトリにのみ保存し、リポジトリへコミットしません。

12. 削除要求

ご自身の投稿がこのツールの取得対象に含まれており、データの削除を希望される場合は、下記「13. 問い合わせ方法」までご連絡ください。具体的な手順はデータ削除案内をご確認ください。

13. 問い合わせ方法

ai.automation.lab.jp@gmail.com

14. 未成年者に関する扱い

本ツールは、投稿者の年齢確認を行わず、未成年者を含む投稿者を意図的に区別する機能を持ちません。未成年者に関する配慮が必要と判断される情報が確認された場合は、個別に削除等の対応を検討します。

15. ポリシー変更

本ポリシーの内容は、運営者の判断により予告なく変更されることがあります。重要な変更がある場合は、本ページの更新をもって周知します。

Privacy Policy

This policy does not assert a specific legal basis for data processing, nor does it declare compliance with any particular law or regulation. The operator will consult a qualified professional as needed before finalizing operational details.

1. Operator

AI副業自動化ラボ (an individually operated project)

2. Service covered

Threads AI副業リサーチBOT — a personally operated market-research tool that uses the official Threads API.

3. Information collected

Through the official Threads API, only the following information is collected.

  • Public Threads posts
  • Post ID
  • Post text
  • Post timestamp
  • Permalink (post URL)
  • Username of the poster
  • Media type
  • Search keywords used
  • Other public fields returned by the official Threads API
  • URLs extracted from post text
  • Classification of extracted URLs (link type)
  • Detected marketing/claim phrases
  • Priority score (calculated by a mechanical, rule-based method)
  • Duplicate-post determination results

4. Information not collected

  • Threads password
  • Instagram password
  • Facebook password
  • Private (non-public) posts
  • Direct messages
  • LINE internal content (chat content, etc.)
  • Full text of paid content such as paid note articles
  • PDF content
  • Personal files stored on a user's device
  • Precise location data
  • Payment information
  • Any credibility score for a poster
  • Any true/false determination of post content
  • Any personal credit or trust evaluation

5. How information is collected

Information is collected only through the official Threads API's public post search functionality. No unofficial methods, web scraping, or browser automation are used.

6. Purpose of use

Collected information is used only for the following purposes.

  • Initial screening, for human review, of trends in AI side-business related public posts, marketing phrasing, and outbound links
  • Calculating a priority score to help prioritize manual review (a mechanical, rule-based aggregation — not a determination of the truthfulness, credibility, or profitability of any post or poster)

Collected information is not used for any purpose beyond the above, including sale or provision to third parties, advertising, or automated posting/actions.

7. Where data is stored

Collected information is stored only on the local file system of an environment managed by the operator. No external database or cloud service is used. Real collected data is never committed to a GitHub repository.

8. Retention period

Collected and generated data is subject to the following maximum retention periods.

Retention period (maximum) by data type
Data typeRetention period (maximum)
Raw JSONUp to 90 days from collection
Processed JSONUp to 180 days from collection
Human-readable JSONUp to 365 days from collection
CSV filesUp to 365 days from collection
Markdown reportsUp to 365 days from collection
LogsUp to 90 days from generation
Access tokensUntil expiration, revocation, disconnection, or no longer needed

These figures are maximum retention periods, not indefinite or permanent storage. No automated deletion system is currently implemented. Retention is currently managed through periodic manual review by the operator, who deletes data once it reaches the applicable limit above. Deletion requests are also processed manually, after confirming the relevant data, as described in "12. Deletion requests" below.

Where required for legal compliance or security-incident response, data may be retained beyond the periods above for the minimum time necessary. This statement does not assert compliance with any specific law or regulation.

9. Third-party sharing

Collected information is not sold, provided, or shared with any third party.

10. External services

This tool does not integrate with any external service other than the official Threads API. This website itself does not load any external scripts, fonts, analytics, or advertising resources.

11. Security measures

  • Access tokens and other credentials are sent only via the Authorization header of API requests.
  • Access tokens and other credentials are never written into source code, logs, generated output, or this website.
  • Collected information is stored only in local directories excluded from Git and is never committed to the repository.

12. Deletion requests

If your post is included in data collected by this tool and you would like it deleted, please contact us using the method in "13. Contact" below. See the Data Deletion Instructions page for the detailed procedure.

13. Contact

ai.automation.lab.jp@gmail.com

14. Minors

This tool does not verify the age of any poster and has no function to intentionally distinguish minors from other posters. If information requiring special consideration regarding a minor is identified, it will be handled, including possible deletion, on a case-by-case basis.

15. Changes to this policy

The content of this policy may change without prior notice, at the operator's discretion. Material changes will be announced by updating this page.
